## Tips

1. **Read the README first** - See what sections exist and where your entry fits
2. **Follow the format** - Match the style of other entries
3. **Alphabetical order** - Add in the right place alphabetically
4. **Be descriptive** - The description helps people understand what it is
5. **Include links** - GitHub and PyPI links help with discoverability
