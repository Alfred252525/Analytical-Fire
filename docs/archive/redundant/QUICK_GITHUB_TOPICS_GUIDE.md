# Quick Guide: Add GitHub Topics (CORRECTED)

## Where to Find It

1. Go to: https://github.com/Analytical-Insider-LLC/Analytical-Fire
2. You're on the "Code" tab (default view)
3. Look at the **right side** of the page, in the row with buttons
4. Find the **"About"** button with a **gear icon** ⚙️
   - It's on the **far right** of that button row
   - Below the repository name
   - Next to buttons like "Code", "Add file", etc.

## How to Add Topics

1. Click the **"About"** button (with gear icon ⚙️)
2. A popup/panel will appear
3. Look for **"Topics"** or **"Add topics"** field
4. Click in that field
5. Type each topic below and press Enter after each:
   - `ai`
   - `artificial-intelligence`
   - `knowledge-sharing`
   - `ai-agents`
   - `collective-intelligence`
   - `ai-platform`
   - `ai-tools`
   - `machine-learning`
   - `ai-collaboration`
   - `python`
   - `fastapi`
   - `ai-analytics`
6. Click **"Save changes"** or click outside to save

## If You Can't Find It

**Alternative method:**
1. Click on **"Settings"** tab (at the top)
2. Scroll down to find **"Topics"** section
3. Add topics there
4. Scroll to bottom and click **"Save changes"**

## Done!

Once saved, topics will appear as tags on the repository page and make it discoverable in GitHub search!
